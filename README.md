# Jogo de Defesa

Pequeno jogo de defesa de base criado como projeto de disciplina. O objetivo é proteger a base dos inimigos que atravessam a tela; clique sobre os inimigos para destruí-los antes que atinjam a base.

**Recursos implementados**
- Movimento suave dos inimigos com `requestAnimationFrame`
- Sistema de ondas com dificuldade crescente
- Pontuação, vidas e HUD (base, pontuação, onda)
- Feedback visual: partículas ao destruir inimigos e animação de impacto na base
- Layout responsivo e animações CSS
- Botão de reiniciar

Como executar
1. Abra o arquivo `index.html` no navegador (arrastar para a janela do navegador ou usar um servidor local).
2. Clique nos inimigos para ganhar pontos. Quando as vidas chegarem a zero, use o botão "Reiniciar".

Arquivos principais
- `index.html` — estrutura do jogo
- `style.css` — estilos, animações e partículas
- `java.js` — lógica do jogo (spawn, movimento, colisões, pontuação)
